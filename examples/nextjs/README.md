# Nextjs

Use `babel-plugin-clsx` in `Nextjs`.

## Install

npm

```bash
npm i babel-plugin-clsx -D
```

yarn

```bash
yarn add babel-plugin-clsx -D
```

pnpm

```bash
pnpm add babel-plugin-clsx -D
```

## Configuration

.babelrc

```json
{
  "presets": [["next/babel"]],
  "plugins": [["clsx", {}]]
}
```
